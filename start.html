Synthesizing tasks for XLogo Mini

Step 1: Check reference task and its solution code

We present one of the reference tasks and its solution code on the left panel of the page. The task is presented below the “Reference Task” header and its solution code is presented under the “Reference Code” header. Carefully view the task and its solution code.

Step 2: Check synthesized Task and its Solution Code

We present a synthesized task and its solution code, for the given reference task, on the right panel. The task is presented below the “Synthesized Task” header and its solution code is presented under the “Synthesized Code” header. Carefully view the task and its solution code.

Step 3: Finish the Survey Questions

You are provided with a google sheet ($NAME_expertFeedback_$DATE.xlxs) with the survey questions to be answered for each synthesized task. Each row in the sheet corresponds to a specific synthesized task. For each task, we present 6 questions. These questions must be answered on a 3-point Likert scale. The rating can be picked from the given dropdown menu in each cell of the google sheet. We describe each question below.

We describe each question below.

Q1: Overall rating of the synthesized task

  • -1 (Poor quality, task does not meet the stated requirements and expectations)
  • 0 (Satisfactory quality, task meets the minimum requirements, but not adequate for students)
  • 1 (Excellent quality, task meets or exceeds all expectations and requirements)

Q2: Difficulty of the synthesized task w.r.t reference task

  • -1 (Easier than the reference task)
  • 0 (Similar difficulty to the reference task)
  • 1 (More difficult than the reference task)

Q3: Rating of synthesized tasks in terms of concepts covered

  • -1 (Misses many important or relevant concepts that are essential)
  • 0 (Contains a few unnecessary or irrelevant concepts that are not included in the reference task.)
  • 1 (The concepts are the same as the reference task)

Q4: If there are distractors (e.g., walls, forbidden objects/grids), are they useful?

  • -1 (Not useful, all the distractors are not helpful for achieving the learning objectives)
  • 0 (Partially useful, some distractors are useful while others may not be useful.)
  • 1 (Very useful, all distractors are essential for achieving the learning objectives)

Q5: Rating of synthesized code w.r.t reference code

  • -1 (Poor quality, synthesized code has many redundant commands and is very confusing)
  • 0 (Acceptable quality, The synthesized code meets the basic requirement, but may contain a few unnecessary or irrelevant commands that could be removed to improve its quality.)
  • 1 (Good quality, the synthesized code includes all necessary commands without any irrelevant ones.)

Guidelines

1. Please make sure that the ID of the task on the webpage matches the ID of the task on the google sheet, while filling in the survey for a specific task.

2. In order to do the survey, it would be more convenient to keep two files open side-by-side: the webpage which describes each task and the google sheet to enter the rating for the task.

3. For some questions, if no rating is valid, please select N/A from the dropdown menu in the sheet. This is the default option for every question.

4. Please complete the survey by X.
