强化学习之Policy Gradient

发表于 2018-01-14 | 分类于 强化学习

Policy Gradient 是一种 Model-free 的方法。

Policy-based Approach: Learning an Actor

首先，我们知道对于深度学习问题，有三步：

  • Define a set of functions.
  • Evaluate the goodness of function.
  • Pick the best function.

下面，我将从这三个角度分别进行说明。

Define a set of functions

在Policy Gradient（PG）方法中，我们把Neural Network（NN）作为actor，这个actor就是我们的策略。这个NN的输入是机器所观察到的环境（比如游戏画面的pixels），而输出层的每一个neuron都对应于一个actor，neuron的值则是这个采取这个action的概率。如下图所示：

Value-based Approach: Learning a Critic

What does a critic do ?

给定一个actor ，critic是用来评估这个actor的优劣的。一般的，actor就是我的策略，而critic就是评估策略的值函数 , 其值为从状态s开始，采取策略 所得到的累积奖赏。

How to estimate the value function?

给定一个actor ，如何估计?主要有两种办法，一种是基于Monte-Carlo的方法，另一种是基于差分的方法。

Monte-Carlo based approach

在基于MC的方法中，critic 看 actor 玩游戏，然后计算每一个episode的累积奖赏。

Temporal-difference approach

有些应用中，一个episode可能会无限长，要延迟到一个episode结束才能够计算total reward，因此无法使用MC来估计。

MC VS. TD

在MC方法中，由于策略的随机性，我们多次计算，然后把所有的结果累加取平均，将这个值作为的估计。MC方法的特点在于最后的评估具有比较大的variance，不过是unbiased的。
